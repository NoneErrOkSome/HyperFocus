## ğŸŒ IP Addressing

Layer 3 uses **logical addresses**:
- IPv4: 32 bits (e.g., 192.168.1.1)
- IPv6: 128 bits (e.g., 2001:0db8::1)

Unlike MAC addresses (Layer 2), IP addresses:
- Can be assigned and changed
- Identify both **network** and **host**

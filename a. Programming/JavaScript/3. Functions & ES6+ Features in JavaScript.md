## **ðŸ“Œ 4. Default Parameters (ES6)**

Set default values if arguments are missing.

```javascript
function greet(name = "Guest") {
    return `Hello, ${name}!`;
}
console.log(greet());   // Output: Hello, Guest!
console.log(greet("Alice"));  // Output: Hello, Alice!
```

---

## **ðŸ“Œ 5. Rest Parameters (`...args`)**

Handles **multiple arguments** as an array.

```javascript
function sum(...numbers) {
    return numbers.reduce((acc, num) => acc + num, 0);
}
console.log(sum(1, 2, 3, 4, 5)); // Output: 15
```

---

## **ðŸ“Œ 6. Spread Operator (`...`)**

Expands arrays or objects into individual elements.

### **Array Example**

```javascript
const nums = [1, 2, 3];
const newNums = [...nums, 4, 5];
console.log(newNums); // Output: [1, 2, 3, 4, 5]
```

### **Object Example**

```javascript
const person = { name: "Alice", age: 25 };
const newPerson = { ...person, country: "USA" };
console.log(newPerson);
// Output: { name: "Alice", age: 25, country: "USA" }
```

## **ðŸ“Œ 7. Higher-Order Functions**

Functions that **take other functions as arguments** or **return functions**.

### **Example: Using `map()`**

```javascript

const squared = numbers.map(num => num * num);
console.log(squared); // Output: [1, 4, 9, 16]
```

## **ðŸ“Œ 8. Function Scope & `this`**

The **value of `this`** depends on how a function is called.

### **Traditional Function (`this` changes)**

```javascript
const obj = {
    name: "Alice",
    greet: function() {
        console.log(`Hello, ${this.name}!`);
    }
};
obj.greet(); // Output: Hello, Alice!
```

### **Arrow Function (`this` is lexically bound)**

Arrow functions **inherit `this` from the surrounding scope**.

```javascript
const obj2 = {
    name: "Alice",
    greet: () => console.log(`Hello, ${this.name}!`)
};
obj2.greet(); // Output: Hello, undefined!
```

âŒ **Arrow functions are not ideal for object methods**

## **ðŸ“Œ 9. Immediately Invoked Function Expressions (IIFE)**

A function that **runs immediately after being defined**.

```javascript

    console.log("IIFE executed!");
})(); // Output: IIFE executed!
```
